# 🎓 Smart Academic Tracker

## Project Overview

**Smart Academic Tracker** is a full-stack web-based academic management system designed to streamline student performance tracking, attendance management, and administrative operations.

The system provides **role-based dashboards** for Admin, Faculty, and Students, along with **AI-powered risk prediction** to identify students who need attention.

---

## Key Features

### Admin Panel

* Create & manage students and faculty
* View analytics dashboard
* Manage examination forms
* Handle student issues
* Publish announcements
* Configure system settings
* Role & permission management

---

### Faculty Panel

* Upload marks
* Upload attendance
* View student records
* AI risk prediction dashboard
* Announcements management
* Performance analytics with charts

---

### Student Panel

* View academic dashboard
* Check marks & attendance
* View performance analytics (charts)
* Download admit cards (PDF)
* Apply exam forms
* Raise issues
* View results (semester-wise)
* Access announcements
* Profile management

---

### AI Features

* Risk prediction based on:

  * Marks
  * Attendance
* Categorization:

  * High Risk 
  * Medium Risk 
  * Low Risk 
* Smart suggestions for improvement

---

## Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* Recharts (for analytics charts)
* Axios (API calls)
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

### Authentication

* JWT (JSON Web Token)
* Role-based access control

---

## System Architecture

Client (React) → API (Express) → Database (MongoDB)
